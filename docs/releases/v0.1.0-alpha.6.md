# v0.1.0-alpha.6

Replacement alpha prerelease for early testers.

Canonical tracked changelog entry: `CHANGELOG.md`

## What's Included

- fix for recognition failures that previously dropped OCR text before
  diagnostics capture
- Diagnostics screen now shows OCR text directly for failed import or
  recognition attempts when that text is available
- local screenshot import, review, and local save on the supported path
- history, detail, dashboard, and diagnostics browsing for saved data and
  support troubleshooting

## Supported Scope

- one supported Simplified Chinese post-match detailed-data screenshot
- local screenshot import
- on-device processing
- required review before final save

Unsupported screenshots are rejected.

## Installation

Download the attached Android release APK from this GitHub prerelease and
install it on your Android device.

If you already installed `v0.1.0-alpha.5`, install this newer APK over it.

If Android blocks installation from the downloaded source, enable installation
for that source and retry.

## Diagnostics Support

If you hit a problem in this build, open `Diagnostics` in the app and copy the
export into your support message. For failed import or recognition attempts,
the diagnostics view and copied export may include OCR text from the failure so
support can inspect what ML Kit actually read. The export still excludes the
original screenshot and full saved record payload.

## Known Limitations

- Hero may still require manual entry during review.
- This is an alpha prerelease, not a stable `1.0.0` release.
- The app does not support additional templates or non-Chinese screenshots in
  this release.

## Privacy

The app is local-first and processes screenshots on-device.
